In Terminal:
	* copy or download "mariadb-java-client-1.5.7.jar" into the same folder
	* compile using:
		javac *.java
	* run using:
		java -cp mariadb-java-client-1.5.7.jar:. ClassName
		
In IntelliJ:
	* copy or download "mariadb-java-client-1.5.7.jar" into the project folder
	* right click on the project within IntelliJ
	* select module settings
	* select Libraries
	* Click plus (+) and select Java
	* Browse and select the above jar file and click OK
	
In Eclipse:
	* copy or download "mariadb-java-client-1.5.7.jar" into the project folder
	* right click on the project within Eclipse
	* select Properties
	* select Java Build Path
	* select Libraries
	* select add JARS
	* Browse and select the above jar file and click OK
	
In NetBeans:
	* copy or download "mariadb-java-client-1.5.7.jar" into the project folder
	* expand the project within NetBeans and right click on Libraries
	* select add JAR/Folder
	* Browse and select the above jar file and click OK
